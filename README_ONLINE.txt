COMO COLOCAR ONLINE

1) Suba estes arquivos no GitHub:
   - app.py
   - requirements.txt

2) No Streamlit Cloud:
   - New app
   - escolha o repositório
   - Main file path: app.py
   - Deploy

3) Para testar no seu computador antes:
   python -m pip install -r requirements.txt
   streamlit run app.py

Observação: esta é a versão web do analisador v72. Ela abre no navegador.
